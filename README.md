# Description
This application allows users to apply different basic filters to their images.
It would be most useful for someone just getting into personal editing, as it explains what each filter does and they can play around with different options.

## Features
This web app allows users to apply the filters of
Image Blur:
Edge Detection:
Image Greyscale:
Sharpen Filter:
Sepia Filter:
Posterize Filter:
Emboss Filter:
Hue Rotation Filter:
